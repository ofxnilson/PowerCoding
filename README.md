# PowerCoding – Duolingo-Style Coding Learning App

An Android learning platform that teaches programming concepts using gamified lessons and progress tracking.

Inspired by Duolingo’s interactive learning path.

---

## Features

- Lesson path visualization (bubble/tree layout)
- Locked / active / completed lessons
- Progress tracking
- RecyclerView dynamic UI
- Backend synchronization via REST API
- Material Design interface
- Responsive mobile layouts

---

## Tech Stack

- Java
- Android SDK
- RecyclerView
- REST APIs
- JSON
- Material 3 UI components

---

## Architecture

- Activity/Fragment based UI
- Lesson model & state management
- API service layer (HTTP/REST)
- Local persistence

---
